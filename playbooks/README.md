# Playbooks

Battle-tested patterns extracted from production projects. Each playbook covers a specific stack with 2-3 code snippets you can copy directly.

These aren't theoretical — every pattern here solved a real problem in a real codebase.

## Available Playbooks

| Playbook | Stack | Key Patterns |
|----------|-------|-------------|
| [Next.js + Supabase](nextjs-supabase.md) | Next.js App Router, Supabase, SWR | BFF routes, `.maybeSingle()`, error states, SWR caching |
| [Python AI/RAG](python-ai-rag.md) | Python, FAISS, LLMs, ElevenLabs | Cache-first pipeline, FAISS queries, voice synthesis |
| [Multi-Agent Systems](multi-agent-system.md) | Claude API, multi-agent orchestration | Model-to-task matching, delegation control, cost rails |

## How to Use

1. Read the playbook for your stack
2. Copy the relevant patterns into your project's `CLAUDE.md`
3. Add project-specific details (table names, API endpoints, etc.)
4. Claude will follow these patterns automatically in future sessions
